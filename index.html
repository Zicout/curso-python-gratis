Instalação do Python

  • Windows

  • Mac

  • Linux

São necessários poucos passos para a instalção do Python no Windows

  1. Vá para o site oficial do Python (https://www.python.org/) e clique no link para download da versão mais recente do Python para o Windows.
  2. Execute o arquivo baixado para iniciar o assistente de instalação.
  3. Siga as instruções na tela para completar a instalação. Certifique-se de selecionar a opção para adicionar Python às variáveis de ambiente do Windows, caso contrário, você precisará especificar o caminho completo para o interpretador Python cada vez que quiser usá-lo.
  4. Depois de instalar, abra o prompt de comando do Windows e digite:

    python --version

    Esse comando deve retornar algo como "Python 3.x.x"

Após concluir a instalação do Python, você precisará de uma IDE ou Editor de Texto para começar a programar. No curso usaremos o Pycharm, mas existem vários outros softwares capazes dessa função.

Instalaremos o Python no Mac através do terminal. Procure o Terminal no Launchpad e inicialize-o

  • Digite o comando abaixo e aperte Y para confirmar os downloads necessários
    • python3
  • Após isso, para chamar o Python basta digitar python3 no console

Normalmente o Linux já possui uma versão do Python 3 inclusa. Você pode verificar a versão instalada em sua distro abrindo o terminal e digitando:

python3 --version

Você deve ver a versão do python sendo 3.10 ou superior. Se ela estiver desatualizada, basta fazer o download com o comando:

sudo apt-get install python3.11

Variáveis

Em Python, uma variável é um nome que é associado a um valor. Você pode usar variáveis ​​para armazenar dados que você deseja usar em seu programa. Para atribuir um valor a uma variável, você usa o sinal de igualdade (=).

É importante notar que, em python, variáveis não precisam ser declaradas antes de serem usadas e também não precisam ser inicializadas com valores específicos, se você atribui um valor a uma variável que não foi criada, ela será criada automaticamente.

Além disso, é importante notar que as variáveis devem ser nomear de forma significativa e coerente, e não podem começar com números ou conter caracteres especiais, exceto o sublinhado.

  • Tipos Primitivos

Considere 4 tipos primitivos:

int (números inteiros): são números inteiros sem casas decimais, como 1, 2, 3, 4, etc. Um exemplo de uso seria:

x = 5
y = -3
z = x + y

float (números de ponto flutuante): são números com casas decimais, como 1.5, 2.7, 3.14, etc. Um exemplo de uso seria:

x = 5.23
y = -3.14

str (strings): são sequências de caracteres, como "hello", "goodbye", "python", etc. Um exemplo de uso seria:

nome = "leandro"
frase = "Eu estou aprendendo python"

bool (valores booleanos): são valores verdadeiro ou falso. Um exemplo de uso seria:

esta_chovendo = True
esta_sol = False


Operações com variáveis

É possível aplicar operações aritméticas com váriaveis números (isto é, variáveis do tipo int e float).

Veja quais são as operações permitidas:

adição = +
subtração = -
multiplicação = *
divisão = /
divisao inteira = //
exponenciação = **
Operações com variáveis do tipo string

É permitido também operações com textos e na verdade chamamos essas operações de "concatenações". Se você somar duas strings (textos) você obtem um texto só. Exemplo:

nome = "leandro"
sobrenome = "duarte"
nome_completo = nome + " " + sobrenome
print(nome_completo)
OUTPUT: >>> leandro duarte

Saída e Entrada de Dados

Para exibir dados em Python, o método mais comum é usar a função print(). Você fez isso com o Hello, World! Lembra?

É possível imprimir variáveis de qualquer um dos tipos que explicamos acima, como também adicionar qualquer Texto, desde que estes estejam entre aspas duplas ou simples.

Não somente exibir, como também coletar dados é possível. E na verdade é aqui que começamos a entender mais sobre como o sistema funciona.

Se print() exibe dados, precisamos de algo que consiga coletar os dados vindo do teclado do usuário, correto? Sim, e para isso utilizamos a função input()

  • Exemplo:
    nome = input("Qual o seu nome: ")
    print(nome)
    OUTPUT:
    >>> Qual o seu nome: leandro duarte
    >>> leandro duarte

O problema de usar o input() sozinho é que todo texto digitado será retornado como uma string. Isto é: se você digitar um número, o computador não faz a conversão automatica do tipo. É preciso informar ao computador qual o tipo de dado você está exigindo do usuário. Se você lembra dos tipos primitivos, tente agora usá-los. Exemplo:

nome = str(input("Me diga o seu nome: "))
idade = int(input("Me diga sua idade:"))

Comece a usar esse tipo de tipagem ao trazer dados, já que em outras linguagens você sempre precisará declarar o tipo de dado antes de solicitá-lo ao usuário. Em java por exemplo, existem funções exclusivas para inteiro, string ou float. É recomendável que vocêmantenha esse hábito para que em outras linguagens não tenha dificuldade


Strings

Sintese

As strings podem ser criadas utilizando aspas simples ou duplas. Exemplo: "Olá, mundo!" ou 'Olá, mundo!'

Como foi supracitado, é possível concatená-las, isto é, somar uma string a outra e assim uni-las.

Em todas as linguagens de programação, é considerado string um conjunto de caracteres que somados formam a palavras, frases e textos. Todos os caracteres podem ser strings, de números a letras e simbolos até espaços vazios.

É comum que em todas as linguagens tenhamos que lidar com esses textos e em python nós podemos trabalhar com elas diretamente através de métodos incluídos na classe str(). Mais pra frente iremos explicar o conceito de classes e métodos mas por agora pense nos métodos como um "meio de manipular um objeto (as string que iremos trabalhar)".

Você agora vai ver os principais métodos que podem ser aplicados em uma string.

len()

  • Contando Caractéres

Principal método para conseguirmos obter a quantidade de caractéres em uma string. Esse método retorna um número do tipo inteiro. Veja o exemplo:

variavel = "oi"
caracteres = len(variavel)
print(caracteres)
OUTPUT:
>>> 2

Vale ressaltar que não lugar da variável você pode colocar uma string entre aspas e ela retornará a quantidade de caractéres normalmente. Outro ponto importante a verificar é que espaços vazios também são considerados caractéres.

lower()

  • Strings minúsculas

As vezes nós queremos deixar um texto todo em minúsculo. Pense naqueles usuários que acabam digitando com CAPS LOCK ativado. Eu particulamente acho muito mais difícil ler um texto todo maiúsculo e também muito mais complexo de se lidar com ele. Vamos utilizar um método chamado "lower()" e seremos capazes de pegar caracteres, palavras, frases e texto e coloca-las todas em minúscula! Exemplo:

variavel = "EU DIGITO COM CAPS LOCK LIGADO"
minuscula = variavel.lower()
print(minuscula)
OUTPUT:
>>> eu digito com caps lock ligado

upper()

  • Quero tudo maiúsculo

As vezes, ao contrário de queremos um dado inserido pelo usuário tudo minúsculo, nós queremos ele todo maiúsculo. Imagine você com um cliente tendo uma loja profissional e o seu cliente está tendo problemas com cupons. Os usuários estão digitando os cupons em minúsculo, mas o sistema só aceita se estiver em maiúsculo. E você vai agora ao seu cliente e diz: EU TENHO A SOLUÇÃO! Eu fiz o curso do Leandro e ele me mostrou que eu posso transformar os dados em maiúsculos! Veja o exemplo:

cupom = "3cmn0p"
maiusculo = variavel.upper()
print(maiusculo)
OUTPUT:
>>> 3CMN0P

Viu como é simples? E acreditem, esses métodos podem parecer bobos mas eles vão acompanhar toda a sua carreira daqui pra frente! Se prepare programador!

strip()

  • Removendo Espaços

Todo mundo conhece aquele usuário que antes de começar a digitar, testa o teclado no formulário apertando varias vezes o espaço. Ou mesmo aquele usuário que sempre deixa um espaço após digitar o email em algum formulário por aí. É importantíssimo tratar esses casos, já que muitas vezes podemos ter exceções (erros) devido a essa má pratica por parte do usuário. Você já deve ter percebido que nós sempre estamos consertando os erros de digitação do usuário né? Faça disso sua rotina, uma vez que o usuário sempre será o maior inimigo dos apps, hahahaha!

Mas voltando.. Entenda que é possível tratarmos esses dados digitados pelos usuários removendo espaços antes e após o termino da palavra, frase ou texto. O método mais eficaz para isso é o strip(). Bora ver um exemplo na prática?

variavel = "    leandro@email.com       "
semEspaco = variavel.strip()
print(semEspaco)
OUTPUT:
>>> leandro@email.com

Repare que antes tinhamos entre os parenteses espaços antes e depois do email. A saída foi a string sem esses espaços e agora ela está completamente tratada. É importante se atentar que esse método não retira os espaços no meio de frases e textos. Para essa função temos outro método que explicarei logo adiante.

replace()

  • Substituindo Caractéres

Existem dois métodos de substitui caractéres dentro de uma string. O mais prático é o que será ensinado agora. O outro método ensinaremos mais a frente quando tratarmos de listas. Lá você também vai abrir a sua mente para o que realmente significa uma string.

O método replace() é extremamente prático e simples. Exemplo:

nome = "leandro duarte"
nomeNovo = nome.replace("duarte", "novaes")
print(nomeNovo)
OUTPUT:
>>> leandro novaes

Perceba que na linha 2 estamos dizendo ao Python: Ei python, procura a palavra "duarte" e troca pra mim pela palavra "novaes". Você percebeu que na verdade nós estamos dizendo ao python o que fazer. Todos esses métodos explicados anteriormente fazem parte de instruções que dizemos ao python, e o python se encarrega de dizer isso ao nosso computador. Mais pra frente entraremos mais a fundo (lá ele) em como esse processo de "python diz ao computador" funciona. Mas não se preocupe por agora, na verdade não é tão difícil!

split()

  • Separando as coisas...

As vezes nos queremos separar um texto em várias partes. Como por exemplo, pedimos ao usuário vários emails e que ele digite cada email separado por uma vírgula.

email = ""email1","email2","email3","email4","email7","email6""
emailSeparado = nome.split(",")
print(emailSeparado)
OUTPUT:
>>> ["email1", "email2", "email3", "email4", "email5", "email6"]

Você deve estar pensando: Poxa, mas não separou não, continua a vírgula ali. Mas eu digo: Calma programador, nós ainda não entendemos esse output (saída) mas iremos tratar sobre em um tópico mais a frente. Mas de antemão vou lhe dizer que isso se trata de uma lista. Basicamente cada email dentro dela é um item só. Essa lista funciona como se fosse uma grande variável onde podemos armazenar item a item dentro dela e acessá-los através de suas posições. Se você ainda não entendeu, fique tranquilo que em um tópico mais a frente eu voltarei a abordar esse método novamente.


Condições

"if" e "else"

  • Criando Possibilidades

Até agora você viu aplicações simples onde nós captamos dados do usuário, tratamos ele e então devolvemos uma saída mostrando esse "dado transformado". Mas e se nós queremos engrandecer o nosso aplicativo através de possibilidades? Imagine que no seu sistema você precise verificar se o usuário é maior de idade ou não. E você imprimirá uma mensagem a ele caso ele seja menor de idade. Existe essa possibilidade e é com if e else que tratamos isso. Veja o exemplo:

idade = int(input("Digite a sua idade: "))

if idade < 18:
    print("Você é menor de idade e não pode acessar o conteúdo do do nosso aplicativo")

else: #caso o usuário seja maior de idade
    print("Seja bem vindo ao nosso sistema!")

Perceba que inserimos informações novas. Você deve se lembrar das aulas de matemática chatas onde utilizávamoso sinal < para representar (menor que) e o sinal > para representar (maior que) Aqui no python nós utilizamos esses sinais da mesma maneira.

Outra coisa importante a se verificar é que após o sinal : nós deixamos a linha identada. Isto é, nós indicamos ao python que aquela linha pertence a instrução anterior, no caso o if idade < 18:

Mas olhando para o código acima percebemos que criamos duas saídas diferente. Vamos analizar então essas duas saídas?

Saída nº 1
OUTPUT 1:
>>> Digite a sua idade: 20
>>> Seja bem vindo ao nosso sistema!

Saída nº 2
OUTPUT 2:
>>> Digite a sua idade: 15
>>> Você é menor de idade e não pode acessar o conteúdo do do nosso aplicativo

Você pode usar quantos if você quiser, entretanto lembre-se que o else deve ser utilizado apenas uma vez a cada if. Para casos onde você precisa testar mais de dois caminhos, o correto será ensinado no próximo tópico.

elif

  • Se não a anterior, tente essa

Um pouco diferente if. O elif só pode ser utilizado se antes ouver um if. Ele serve como um segundo if. Você pode colocar quantos elif você quiser. Veja o exemplo:

numero = 10

if numero < 10:
    print("É menor que 10")

elif numero == 10:
    print("É igual a 10")

else:
    print("É maior que 10")

OUTPUT: >>> É igual a 10
